
***Idea Vault 💡***

Idea Vault is a Flask-based web application designed to store and manage ideas in a simple and structured way. It allows users to quickly save their thoughts in one central place.

Currently, the app focuses on adding ideas, with more interactive features planned for future updates.

**🚀 Current Features**

Add new ideas

Simple and clean user interface

Backend integration for storing ideas

**🔮 Upcoming Features**

The app is still expanding. Planned features include:

Upvote system

Comment section

Delete ideas

Edit ideas

User authentication

Improved UI/UX

**🛠️ Tech Stack**

Python

Flask

HTML / CSS

Supabase (Backend & Database)

Supabase is used to handle backend services and data storage.

**📦 Installation**

Clone the repository:

git clone https://github.com/Jeetmakani51/IdeaVault
cd IdeaVault

Create a virtual environment:

python -m venv env

Activate the virtual environment:

*Windows*

env\Scripts\activate

*Mac/Linux*

source env/bin/activate

Install dependencies:

pip install -r requirements.txt

Run the application:

python app.py

The app will run on:

http://127.0.0.1:5000/
